A fully responsive and modern personal portfolio website showcasing services, resume, skills, portfolio projects, and contact form. Designed and developed using HTML, CSS, and JavaScript to create a professional web presence.

## Features

- **Fully Responsive Design**
- **Animated Section Transitions**
- **Interactive Navigation Menu**
- **Resume Section with Toggleable Tabs**
- **Portfolio Showcase with Live Project Links**
- **Contact Form**
- **Modern UI/UX Design**

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nishatrhythm/Complete-Responsive-Personal-Portfolio-Website-Using-HTML-CSS-JavaScript.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Complete-Responsive-Personal-Portfolio-Website-Using-HTML-CSS-JavaScript
   ```
3. Open `index.html` in your browser.

## Customization

- **HTML (`index.html`)**: Modify the content (name, skills, projects, etc.) based on your profile.
- **CSS (`style.css`)**: Customize colors, fonts, or layout to match your personal style.
- **JavaScript (`script.js`)**: Enhance interactivity or add new animations.
- **Images**: Replace placeholder images in the `images` folder with your own.
